## [1.0.0] — 🤩 Initial Release

### Added
- Result calculator with multi-subject marks entry
- Subject dropdown: English, Urdu, Maths, Computer, Chemistry, Islamiyat, Physics, Statistics, Biology, Other (custom name)
- Auto-calculation of per-subject percentage and letter grade
- Overall percentage, grade, and pass/fail status
- 3-step result creation flow: Name → Subjects → Preview & Save
- Save results to localStorage with persistence across sessions
- Dashboard grid view of all saved results
- Result detail view with full subject breakdown table and progress bars
- Edit existing results
- Delete results with confirmation prompt
- Inline result renaming from detail view
- Pakistani grading scale (A+ at 90% down to F below 40%)
- Pass/fail logic: requires ≥ 40% in every subject and overall
- Dark mode glassmorphism UI — deep navy gradient, electric blue accent
- Sora + Outfit typography
- Fully responsive layout (mobile, tablet, desktop)
- Zero dependencies — single HTML file
