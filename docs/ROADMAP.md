# Roadmap — Gradix

Items are loosely ordered by priority. Nothing here has a deadline.

---

## Near-term

- [ ] **PDF / image export** — Generate a shareable report card from any saved result
- [ ] **Target grade calculator** — Enter a desired grade and see what marks you need to achieve it
- [ ] **Full-text search** — Search across all saved results by name or subject
- [ ] **Light theme** — An optional light mode for daytime use

---

## Medium-term

- [ ] **Spaced repetition review reminders** — Prompt the user to review results after a set period
- [ ] **Semester view** — Group multiple results under a semester and calculate a combined GPA/percentage
- [ ] **Subject performance trends** — Chart how a specific subject's score has changed across results
- [ ] **Import / export JSON** — Back up all data or move it to another device

---

## Long-term / Exploratory

- [ ] **Optional cloud sync** — Encrypted sync across devices without accounts (e.g. via a passphrase)
- [ ] **PWA with offline install** — Full service worker implementation and installable manifest
- [ ] **Grade goal setting** — Set a target percentage per subject and track progress toward it
- [ ] **Multi-language support** — Urdu interface option

---

## Completed

- [x] Result calculator with per-subject marks entry
- [x] Auto grading (A+, A, B, C, D, E, F)
- [x] Pass / Fail detection (per-subject and overall)
- [x] Save, edit, and delete results via localStorage
- [x] Inline result renaming
- [x] Subject dropdown with custom "Other" option
- [x] Mobile-responsive layout
- [x] Dark mode glassmorphism UI
